Velora Hotels

📌 Description

Velora Hotels est un site web moderne d’hôtel développé en HTML et CSS.
Il présente les services, les chambres et l’ambiance d’un hôtel haut de gamme à travers une interface élégante et responsive.

✨ Fonctionnalités

- Page d’accueil avec présentation du concept
- Galerie d’images des chambres et espaces
- Section services (piscine, restaurant, etc.)
- Formulaire de réservation (statique)
- Design moderne et responsive

🛠️ Technologies utilisées

- HTML5
- CSS3

🚀 Objectif du projet

Ce projet a été réalisé dans le but de :

- pratiquer le développement web (HTML/CSS)
- créer un site vitrine professionnel
- construire un portfolio de développeur

🔮 Améliorations futures

- Ajout de JavaScript pour rendre le site dynamique
- Système de réservation fonctionnel
- Animations avancées
- Connexion à une base de données
